Connect to Firebase for real-time sync between two devices. One-time setup takes about 3 minutes.

1
Go to console.firebase.google.com and create a new project (any name). Disable Google Analytics when asked.
2
Click Build → Realtime Database → Create database → choose any location → start in test mode.
3
Go to Project Settings (gear icon) → scroll to "Your apps" → click the </> Web icon → register app → copy the config values below.
4
Paste your values, hit Connect, then share this same HTML file with the other person — you'll both see live updates.
